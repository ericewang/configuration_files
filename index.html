configuration_files

Personal repository for keeping track of development tools used. Contains vim plugins and configuration files for terminal-invoked programs.

Personal config files:

  • .vimrc
  • .tmux.conf
  • .bashrc
  • .gitconfig

Vim plugins (all installed via pathogen.vim):

  • AG (note requires "make install" to create)
  • Ctrl-P
  • Supertab
  • Vim-sensible
  • Vim-scala
  • Vim-fugitive
